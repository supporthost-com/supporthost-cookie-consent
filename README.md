Easily set up cookie bar, categorize cookies by categories, keep a consent report and have your website fully compliant with GDPR, RGPD, DSGVO.

== Description ==

SupportHost Cookie Consent plugin will help you  making your website GDPR (RGPD, DSVGO) compliant.

This plugin doesn't have a pro version, this means that all the features will be available for free for everyone. If you want to support the plugin consider using <a href="https://supporthost.com/supporthost-cookie-consent/">SupportHost web hosting services</a>.

= Key Features =

- Add your scripts using our categories, you can use the plugin to add the scripts or use any other method you prefer

- The plugin blocks the scripts until the cookies for a certain category are accepted by the user.

- The plugin shows a cookie bar, you can decide the style, theme and position.

- The plugin shows a modal windows, for the user to revoke his consent to all cookies or only certain catgories. The modal is also used to show a cookies table, where you can explain which cookies are used on the website and what is their expiry date, domain and function.

- If the user revokes the consent, the cookies are deleted.

- You can fully customize all text for your website.

- The plugin has an "admin-only" mode, so that while you configure the plugin, the cookie bar is only shown to administrators.

- The plugin has 2 shortcodes. 1 to show the modal window, one to add a link/button to delete all cookies.

- Works with every cookie and every script, as long as the scripts are added to your website the right way. For a complete guide of SupportHost cookie consent check <a href="https://supporthost.com/supporthost-cookie-consent/">this guide</a>.

- Pick different styles and positions for the cookie bar and the modal setting.

- Integrates with Polylang.

- Consent report of anonymized IPs, to keep a record of who accepted which cookies and when. You can identify the user if needed using the value of accepted-cookie-consent-id cookie, to delete the user from your records.




= SupportHost cookie consent Plugin Uses The Following Cookies =

- accepted-cookie-consent-id - this is the primary cookie. It stores a random id,    that is used to identify the user in the consent report.

- cc_cookie - This cookie stores the user preferences: which categories of cookies have been accepted?

== Installation ==

= Automatic installation =

Installing the plugin automatically is the easiest way to install the plugin. You can install it with 2 clicks directly from your WordPress dashboard.


From your admin dashboard, go to Plugins > Add New. From the search box, type “SupportHost Cookie consent” and search. Click the install button on the SupportHost Cookie Consent and then activate the plugin.

= Manual installation =

You can also install the plugin manually. Just download the zip file of the plugin from the WordPress.org plugin repository. You can upload the zip file from you WordPress dashboard, or unzip it on you computer and upload the files inside the wp-content/plugins directory using an FTP client.

When doing a manual installation

1.  Upload the plugin folder (unzipped) to the /wp-content/plugins/ directory.
2.  Activate the plugin through the Plugins page in the WordPress dashboard.

= First configuration =

1. Go to settings -> SupportHost cookie consent
2. Fill all texts and cookies table for defined languages (Polylang plugin is supported)
3. Add the cookies your website uses in the form. More info on the <a href="https://supporthost.com/supporthost-cookie-consent/">guide here</a>
4. From the *Settings* tab you can set your Google Tag Manager ID, if you use Google Tag Manager. You can also add your scripts in the header and footer.
5. From the *Appearance* tab you can change the appearance of the cookie bar and cookie modal. The options should be self explainatory, in case check the <a href="https://supporthost.com/supporthost-cookie-consent/">guide here</a>

= Plugin updates =

When an update for the plugin will be available, you will be notified in the WordPress dashboard. You can directly update the plugin from your dashboard. We recommend that you keep the latest version of the plugin to be able to use the latest functionalities.
